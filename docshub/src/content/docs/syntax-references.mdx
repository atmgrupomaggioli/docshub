  description="Example endpoint description"
  label="STABLE"

  description="Example endpoint description"

## Example Alert Docs

<Alert type="info">Example message</Alert>
<Alert type="error">Example message</Alert>
<Alert type="success">Example message</Alert>
<Alert type="warning">Example message</Alert>

- Usage:

```tsx
<Alert type="warning">Example message</Alert>
```

- `type` properties:

```ts
export type AlertType = "success" | "error" | "warning" | "info";
```

### Heading

Use `#` to define different levels of headings (from h1 to h4).

```markdown
## Heading Level 1

### Heading Level 2

#### Heading Level 3
```

### **Bold** _Italic_ ~~Strikethrough~~

**Bold**
